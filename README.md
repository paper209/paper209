## Projects
* **[lkm-outbound-filter](https://github.com/paper209/lkm-outbound-filter)**: It works on the Linux Kernel, using Netfilter to filter outbound traffic.
* **[outbound-filter](https://github.com/paper209/outbound-filter)**: Parses filter rules and sends them to the lkm-outbound-filter over UDP.
* **[icmp-protocol](https://github.com/paper209/icmp-protocol)**: A custom protocol based on ICMP tunneling.
* **[tor-connect](https://github.com/paper209/tor-connect)**: A Tor based persistent client network that distributes validated proxy updates to connected clients.
